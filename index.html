Apa Yang Telah saya Pelajari

Dalam aktivitas pembelajaran pada mata kuliah Pemrograman WEB terdapat beberapa materi yang saya pelajari seperti :

HTML5 & CSS3

Dengan HTML kita dapat mengelola serangkaian data dan informasi sehingga suatu dokumen dapat di akses dan ditampilkan di internet melalui layanan web.

JavaScript

Dengan JavaScript pada HTML kita menggunakanya dalam pengembangan sebuah website agar website yang kita buat dapat lebih dinamis dan interaktif.

PHP

Dengan PHP kita dapat mengembangkan website, baik website statis maupun website dinamis dengan banyak segudang fitur pendukung dan memiliki fleksibilitas yang tinggi.

Mysql

Dengan Mysql yang merupakan perangkat lunak berbasis web server yang bersifat open source, digunakan sebagai localhost yang berfungi pada proses pengeditan desain, dan pengembangan aplikasi.
